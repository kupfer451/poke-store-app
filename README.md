# poke-store-app
Tienda de cartas Pokémon en formato móvil

## Configuración del Proyecto (Android Nativo con Kotlin)

*   **Creación del Proyecto**: Se creó un proyecto de Android utilizando Android Studio.
*   **Lenguaje**: Se eligió **Kotlin** como el lenguaje de programación principal. Kotlin es el lenguaje recomendado por Google para el desarrollo de Android, ya que es moderno, conciso y seguro.
*   **Sistema de Construcción**: Se configuró el sistema de compilación **Gradle** (`build.gradle.kts`) para gestionar las dependencias y la configuración del proyecto de manera eficiente utilizando Kotlin DSL.
